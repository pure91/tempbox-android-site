개인정보처리방침

잠깐함은 잠깐 필요한 사진, 메모, 번호를 사용자의 Android 기기 안에 임시 보관하는 로컬 우선 앱입니다. 잠깐함 앱은 사용자의 개인정보를 수집하지 않습니다.

1. 수집하는 개인정보

잠깐함은 개인정보를 수집하지 않습니다.

  • 회원가입을 받지 않습니다.
  • 로그인을 제공하지 않습니다.
  • 서버 계정을 만들지 않습니다.
  • 광고 SDK와 분석 SDK를 사용하지 않습니다.
  • AI 기능과 원격 설정을 사용하지 않습니다.

2. 서버 전송 여부

잠깐함은 사용자가 앱에 저장한 사진, 메모, 번호를 개발자 서버로 전송하지 않습니다. 앱은 INTERNET 권한을 사용하지 않습니다.

3. 앱 안에 저장되는 정보

사용자가 직접 입력하거나 선택한 임시 항목은 사용자의 기기 안에 있는 앱 전용 내부 저장소에 저장됩니다. v1 기준 메타데이터는 JSON 파일로 저장되며 Room 또는 SQLite를 사용하지 않습니다.

저장된 항목은 사용자가 정한 만료 시간 이후 WorkManager 기반 정리 작업으로 삭제됩니다. 사용자는 앱 안에서 항목을 직접 삭제할 수 있습니다.

4. 사진 접근 방식

사진은 Android Photo Picker를 통해 사용자가 명시적으로 선택한 항목만 접근합니다. 잠깐함은 전체 파일 접근 권한이나 갤러리 전체 접근 권한을 사용하지 않습니다.

5. 사용하지 않는 권한

잠깐함은 다음 권한을 사용하지 않습니다.

  • INTERNET 권한
  • 위치 권한
  • 연락처 권한
  • 전체 파일 접근 권한
  • SMS 권한
  • 접근성 서비스

6. 제3자 제공과 분석

잠깐함은 사용자의 앱 데이터를 제3자에게 제공하지 않습니다. 광고 SDK, 분석 SDK, 원격 설정 SDK를 포함하지 않습니다.

7. 결제

유료 기능은 Google Play Billing의 `pro_lifetime` 비소모성 일회성 상품으로 제공될 수 있습니다. 개발자는 앱 안에서 신용카드 번호 같은 결제 수단 정보를 직접 수집하거나 저장하지 않습니다. 결제 처리는 Google Play 결제 시스템을 통해 이루어집니다.

8. 문의

개인정보처리방침과 앱 지원 문의는 공개 GitHub 저장소 를 통해 확인할 수 있습니다.
