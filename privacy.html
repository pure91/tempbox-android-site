잠깐함
소개 개인정보 지원

최종 업데이트: 2026년 7월 6일

개인정보처리방침

잠깐함은 잠깐 필요한 정보만 사용자의 기기 안에 보관하는 로컬 우선 앱입니다.

수집하는 정보

잠깐함은 계정, 이메일, 전화번호 같은 개인정보 식별 항목을 요청하거나 수집하지 않습니다.

저장 위치

앱 데이터는 사용자의 기기에 로컬로 저장됩니다. 잠깐함은 서버로 전송하지 않습니다.

사진은 Android Photo Picker로 사용자가 직접 선택한 항목만 접근합니다.

권한 안내

INTERNET 권한을 사용하지 않습니다. 위치, 연락처, 전체 파일 접근, SMS, 접근성 서비스 권한도 사용하지 않습니다.

자동 삭제 정책

사용자가 설정한 보관 기간이 지나면 앱 안의 해당 항목을 정리합니다.

문의처

개인정보 관련 문의는 GitHub 저장소 를 통해 확인할 수 있습니다.
