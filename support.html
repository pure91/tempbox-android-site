Support

지원

잠깐함은 서버, 로그인, 인터넷 권한 없이 동작하는 Android 로컬 우선 앱입니다. 문의와 이슈 확인은 현재 공개 GitHub 저장소를 통해 안내합니다.

GitHub 저장소 열기

앱 데이터는 어디에 저장되나요?

사용자가 저장한 임시 사진, 메모, 번호는 앱 전용 내부 저장소에 로컬로 저장됩니다.

앱 데이터를 서버로 보내나요?

아니요. 잠깐함은 서버가 없고 `INTERNET` 권한을 사용하지 않습니다.

사진 전체에 접근하나요?

아니요. Android Photo Picker로 사용자가 선택한 사진만 접근합니다. 전체 파일 접근 권한을 사용하지 않습니다.

저장한 항목은 어떻게 지우나요?

앱 안에서 직접 삭제하거나, 사용자가 정한 만료 시간이 지나면 WorkManager 기반 정리 작업으로 삭제됩니다.

구독이 있나요?

구독 없음. 유료 기능은 `pro_lifetime` 평생권 1회 구매 방식으로 제공될 예정입니다.

환불은 어디서 처리하나요?

Google Play에서 구매한 항목의 환불은 Google Play의 구매 내역과 정책 기준으로 처리됩니다.
